Elementos de Citacoes

The blockquote element

   Entendendo sobre a Tag <blockquote>:

A <blockquote> e usada em uma sessao para citar um fonte cujo foi retirada as informacoes de um a outra sessao.

Nesta secao esta sendo utilizado a tag <blockquote>, usando o atributo cite, indicando o site w3schools,
de onde esta sendo retirado as informacoes passadas por essa sessao

The cite element

De acordo com MDN a tag <cite> representa uma referencia de um trabalho artistico deve incluir um titulo do trabalho ou um URL de referencia.

The quote element

O elemento quote - <quote> - e destinados para citacoes curtas que nao requerem quebra de paragrafos ou quebra de linha. -- MDN Page.
